Guess My Number

A number guessing game built with HTML, CSS, and JS technologies. In this game, the user is required to choose a number between 1 and 20 and begin checking if there is a match between the selected number and the computer-generated random number. The game starts with the user having 100 points, and each mismatch deducts 10 points from the user. Upon a successful match, the user's points are saved, so clicking "Play Again" displays the user's high score, challenging them to surpass the displayed score. The game is interactive and presents various displays on the browser upon success or failure.
