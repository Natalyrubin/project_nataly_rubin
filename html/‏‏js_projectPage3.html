Rock, Paper, Scissors

A Rock, Paper, Scissors game built with HTML, CSS, and JS technologies. The classic game played between the user and the computer, where the user selects one of the options, and the computer randomly chooses. After 10 rounds of choices by the user, the winner is determined based on the highest score. The game is a simple yet engaging implementation of the timeless Rock, Paper, Scissors concept.
