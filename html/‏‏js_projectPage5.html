E-com WEB

A mini e-commerce site with a designated area for site administrators, built using HTML, CSS, and JS technologies. In this project, users can explore a mini-market where they have the option to browse a variety of products and add them to the cart. The cart calculates the total value of the items and displays them on a separate page. Each item has a dedicated page providing specific details about the product, and the cart dynamically updates and tracks changes automatically. Additionally, the area designated for the site administrator requires entry of a code and username to gain authorization. After logging in, the admin can edit, modify, and delete items from the database. The distinct areas for shoppers and administrators are designed to enhance the user experience tailored to each user's needs.
